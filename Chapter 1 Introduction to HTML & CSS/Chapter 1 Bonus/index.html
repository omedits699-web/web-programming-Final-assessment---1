HTML and CSS Resource Bonus

This guide provides a quick reference to essential HTML tags and CSS properties for web development.

HTML Tags

Below is a list of commonly used HTML tags, their purposes, and examples.

p - Paragraph

Defines a paragraph of text.

p {
    margin-bottom: 1em;
}

a - Anchor/Link

Creates a hyperlink to another page or resource.

a

img - Image

Embeds an image in the document.

img {
    width: 100%;
    height: auto;
}

ul and li - Unordered List

Creates a bulleted list.

ul {
    list-style-type: disc;
}



li {
    margin-bottom: 0.5em;
}

ol and li - Ordered List

Creates a numbered list.

ol {
    list-style-type: decimal;
}




li {
    margin-bottom: 0.5em;
}

div - Division

A generic container for grouping content.

div {
    margin-bottom: 1em;
}

span - Inline Container

An inline container for styling or scripting.

p {
    color: blue;
}

strong - Strong Emphasis

Makes text bold to indicate importance.

p {
    font-weight: bold;
}

em - Emphasis

Makes text italic to indicate emphasis.

p {
    font-style: italic;
}

br - Line Break

Inserts a line break.

p {
    margin-bottom: 1em;
}

hr - Horizontal Rule

Creates a horizontal line to separate content.

p {
    margin-bottom: 1em;
}

CSS Properties

Essential CSS properties for styling web pages.

color

Sets the color of text.

p {
    color: blue;
}

background-color

Sets the background color of an element.

body {
    background-color: lightgray;
}

font-family

Specifies the font family for text.

body {
    font-family: Arial, sans-serif;
}

font-size

Sets the size of the font.

h1 {
    font-size: 2em;
}

margin

Sets the margin around an element.

p {
    margin: 10px;
}

padding

Sets the padding inside an element.

div {
    padding: 20px;
}

border

Sets the border around an element.

img {
    border: 1px solid black;
}

text-align

Aligns text within an element.

h2 {
    text-align: center;
}

width and height

Sets the width and height of an element.

img {
    width: 300px;
    height: 200px;
}

display

Specifies the display behavior of an element.

.inline-block {
    display: inline-block;
}

End of Guide

For more information, refer to official documentation.
