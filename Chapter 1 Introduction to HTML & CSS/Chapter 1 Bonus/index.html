HTML and CSS Resource Guide

This guide provides a quick reference to essential HTML tags and CSS properties for web development students.

HTML Tags

Below is a list of commonly used HTML tags, their purposes, and examples.

h1 to h6 - Headings

Headings are used to define the structure and hierarchy of content. h1 is most important, h6 is least.

<h1>Main Heading</h1>
<h2>Subheading</h2>
<h3>Sub-subheading</h3>

p - Paragraph

Defines a paragraph of text.

<p>This is a paragraph of text.</p>

a - Anchor/Link

Creates a hyperlink to another page or resource.

<a href="https://example.com">Click here</a>

img - Image

Embeds an image in the document.

<img src="image.jpg" alt="Description">

ul and li - Unordered List

Creates a bulleted list.

<ul>
    <li>First item</li>
    <li>Second item</li>
</ul>

ol and li - Ordered List

Creates a numbered list.

<ol>
    <li>First step</li>
    <li>Second step</li>
</ol>

div - Division

A generic container for grouping content.

<div class="container">
    <p>Content here</p>
</div>

span - Inline Container

An inline container for styling or scripting.

<p>This is <span class="highlight">highlighted</span> text.</p>

strong - Strong Emphasis

Makes text bold to indicate importance.

<p>This is <strong>important</strong> text.</p>

em - Emphasis

Makes text italic to indicate emphasis.

<p>This is <em>emphasized</em> text.</p>

CSS Properties

Essential CSS properties for styling web pages.

color

Sets the color of text.

p {
    color: blue;
}

background-color

Sets the background color of an element.

body {
    background-color: lightgray;
}

font-family

Specifies the font family for text.

body {
    font-family: Arial, sans-serif;
}

font-size

Sets the size of the font.

h1 {
    font-size: 2em;
}

margin

Sets the margin around an element.

p {
    margin: 10px;
}

padding

Sets the padding inside an element.

div {
    padding: 20px;
}

border

Sets the border around an element.

img {
    border: 1px solid black;
}

text-align

Aligns text within an element.

h2 {
    text-align: center;
}

width and height

Sets the width and height of an element.

img {
    width: 300px;
    height: 200px;
}

display

Specifies the display behavior of an element.

.inline-block {
    display: inline-block;
}

End of Guide

For more information, refer to official HTML and CSS documentation.
