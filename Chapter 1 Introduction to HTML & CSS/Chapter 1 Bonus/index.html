CSS Properties

Essential CSS properties for styling web pages.

color

Sets the color of text.

p {
    color: blue;
}

background-color

Sets the background color of an element.

body {
    background-color: lightgray;
}

font-family

Specifies the font family for text.

body {
    font-family: Arial, sans-serif;
}

font-size

Sets the size of the font.

h1 {
    font-size: 2em;
}

margin

Sets the margin around an element.

p {
    margin: 10px;
}

padding

Sets the padding inside an element.

div {
    padding: 20px;
}

border

Sets the border around an element.

img {
    border: 1px solid black;
}

text-align

Aligns text within an element.

h2 {
    text-align: center;
}

width and height

Sets the width and height of an element.

img {
    width: 300px;
    height: 200px;
}

display

Specifies the display behavior of an element.

.inline-block {
    display: inline-block;
}
